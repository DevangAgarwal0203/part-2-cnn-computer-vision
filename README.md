# CNN Computer Vision Project

## Surface Defect Classification using CNN

This project implements a Convolutional Neural Network (CNN) for image classification of surface defects.

## Dataset Categories

- Normal
- Dent
- Scratch
- Stain

## Tasks Performed

1. Problem Identification
2. Dataset Exploration
3. Image Preprocessing
4. CNN Model Building
5. Model Training
6. Model Evaluation
7. Predictions and Performance Analysis

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Performance

The CNN model was trained on image data and evaluated using:
- Accuracy
- Loss
- Confusion Matrix
- Classification Predictions

## Output Files

Results generated:
- Class distribution graph
- Sample images
- Confusion matrix
- Training performance graph
- Sample predictions
